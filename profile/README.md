## Featured Project

### [`bnfn`](https://github.com/EasyDeployDev/bnfn)

Bun-native HTTP client with an Axios-shaped API and a required Zig acceleration layer.

- Bun runtime only
- Zig-backed native primitives
- Axios-shaped request flow
